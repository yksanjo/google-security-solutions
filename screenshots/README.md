## How to Take Screenshots

1. **Chrome Monitor**: Start server (npm start), open http://localhost:3000, take screenshot
2. **Extension Analyzer**: Start server (npm start), open http://localhost:3001, take screenshot
3. **AI Defender**: Run tests (npm test), capture terminal output
4. **Supply Chain**: Run scan (python scan.py), capture terminal output
5. **Android Scanner**: Run scan (python scan.py --device), capture terminal output
